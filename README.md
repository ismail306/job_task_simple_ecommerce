## About This Taks

I have completed the e-commerce platform task as per the project requirements. The project includes four seeders for Category, SubCategory, Product, and User. You can either import the attached SQL file into your database or run the seeders to access the Admin dashboard and test the CRUD functionality across all modules.

Only login functionality is available, as I used Laravel Breeze for authentication. Additionally, I implemented interfaces and repositories for the Category module only, due to time constraints.
